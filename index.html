Projects

Weather Monitor

This compact system uses NodeMCU ESP8266 and sensors to monitor weather conditions and send data to Blynk. It enables remote monitoring and automated actions, making it ideal for smart farming and home automation..

Earthquake Detector

This system uses an Arduino Nano, accelerometer, LCD, and buzzer to detect seismic activity. When vibrations exceed a set threshold, it triggers an alarm and displays data, offering an efficient early warning for earthquake preparedness.

Parking Assistant

This Arduino-based system uses IR sensors to detect vehicles, triggering a servo motor to lift a barrier and a relay to control lights or indicators. It offers an efficient, automated solution for managing parking spaces with minimal manual effort.
